abstract = "The manufacturing industry employs legacy equipment for an economical production. Especially old machine tools and programmable logic controllers lack network connectivity and standardized protocol support. With the advent of the fourth industrial revolution those machines need to be integrated with existing infrastructures and large scale Cyber-Physical Production Systems (CPPS). Due to the complexity of modern fabrication, hierarchies for distributed monitoring and control need to be established. We suggest the enclosure of the legacy machine's state and functionality into a surrogate computation model for conquering those issues. Standardized information and communication protocols ensure seamless vertical and horizontal integration, whereas cyber-physical feedback enables adaptive machine behavior for improved automation. The proposed software-framework builds a bridge between heterogenous IoT middlewares and the superior CPPS in a flexible manner."
